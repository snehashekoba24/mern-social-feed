🌄 MERN Social Feed

A simple MERN stack app where users can upload images with captions and view them in a feed.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------
✨ Features

Upload image with caption
Store images in ImageKit
Save image URL in MongoDB
Display posts in a feed

🛠 Tech Stack

Frontend: React (Vite)
Backend: Node.js, Express
Database: MongoDB
Cloud Storage: ImageKit
